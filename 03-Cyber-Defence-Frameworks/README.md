
# Module 03 – Cyber Defence Frameworks

## Overview

This module introduces several defensive security frameworks that help Security Operations Center (SOC) analysts understand attacker behavior, prioritize detections, and improve incident response.

Rather than focusing on individual security tools, these frameworks provide structured ways to think about how attackers operate and how defenders can disrupt or detect malicious activity.

---

## Learning Objectives

After completing this module, I can:

- Explain the Pyramid of Pain and why higher-level detections are more effective.
- Describe the stages of the Cyber Kill Chain.
- Understand the purpose of the Unified Kill Chain.
- Navigate the MITRE ATT&CK framework.
- Apply defensive frameworks during security investigations.

## Why These Frameworks Matter

Modern SOC analysts investigate incidents by understanding attacker behaviour rather than simply reacting to alerts.

Frameworks such as MITRE ATT&CK and the Pyramid of Pain help analysts answer questions like:

- What is the attacker trying to achieve?
- Which stage of the attack are we observing?
- How can we detect similar attacks in the future?
- Which detections will have the greatest impact?
