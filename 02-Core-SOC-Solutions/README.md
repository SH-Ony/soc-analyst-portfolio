
# Module 02 – Core SOC Solutions

## Overview

This module introduces the core technologies used by Security Operations Centers (SOCs) to detect, investigate, and respond to cyber threats.

Unlike Module 01, which focused on SOC workflows and analyst responsibilities, this module focuses on the security platforms that analysts use every day. These include SIEM, EDR, SOAR, and log analysis tools such as Splunk and the Elastic Stack.

Understanding how these technologies work together is essential for effective security monitoring and incident response.

---

## Learning Objectives

After completing this module, I can:

- Explain the purpose of SIEM, EDR, and SOAR.
- Understand how security logs are collected and analyzed.
- Perform basic investigations using Splunk.
- Understand the fundamentals of the Elastic Stack.
- Explain how different SOC technologies work together during incident response.

## Real World Connection

In a real Security Operations Center, analysts rarely use a single security platform.

Instead, SIEM, EDR, SOAR, firewalls, IDS/IPS, Active Directory, and threat intelligence platforms all work together to detect and investigate security incidents.

This module provided the foundation for understanding how these technologies interact throughout the incident response lifecycle.
